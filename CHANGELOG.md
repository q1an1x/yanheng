# 更新记录

## 0.2.0 - 2026-07-17

- 删除言衡应用专属 Swift 与伴生动态库，API、配置、定时与统计全部改为纯言序实现。
- 修复应用图标白色方形背景，发布 PNG 现在具有真实透明边缘。
- 修复数值 Stepper 不显示当前值，并将默认最少可用账号改为 1。
- 从账号 `extra` 正确读取 Codex 与 Anthropic 的 5h/7d 用量窗口。
- 新增今天、7 天、总计周期切换，以及模型分布、用户消费榜、请求明细、SLA 和实时 QPS/TPS。
- Token 输入、输出、缓存读写、请求、消费和总量均使用独立 SF Symbol 图标。

## 0.1.0 - 2026-07-17

- 首个公开版本。
- 原生 macOS 菜单栏应用，基于 `yanxu-macos-ui`。
- 聚合 sub2api 账号可调度状态、5h/7d 平均余量与补充预警。
- 展示今日输入、输出、缓存读写 token，以及 RPM、TPM、请求和成本。
- 管理员 API Key 存入 macOS Keychain，支持自动与手动刷新。
